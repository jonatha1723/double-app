# Compilar APK no Celular - App Double

Siga este guia para compilar o app Double em APK direto no seu celular.

## Passo 1: Criar Conta Expo (Gratuita)

1. Acesse https://expo.dev no seu celular
2. Clique em "Sign Up" (Criar Conta)
3. Preencha com:
   - Email
   - Senha
   - Nome de usuário
4. Confirme o email

## Passo 2: Acessar o Projeto

1. Vá para https://expo.dev/projects
2. Clique em "New Project"
3. Escolha "Import from GitHub" ou "Upload Project"

### Se usar GitHub:
1. Conecte sua conta GitHub
2. Selecione o repositório do projeto Double
3. Clique em "Import"

### Se fazer upload:
1. Baixe o arquivo ZIP do projeto
2. Clique em "Upload"
3. Selecione o arquivo ZIP

## Passo 3: Configurar Build

1. No dashboard do projeto, clique em "Build"
2. Clique em "Create New Build"
3. Selecione "Android"
4. Escolha o tipo de build:
   - **APK** (para instalar diretamente)
   - **AAB** (para Play Store)

## Passo 4: Esperar a Compilação

1. A compilação começará automaticamente
2. Você verá o progresso em tempo real
3. Levará entre 10-30 minutos
4. Você receberá uma notificação quando estiver pronto

## Passo 5: Baixar o APK

1. Quando a compilação terminar, clique em "Download"
2. O APK será baixado para seu celular
3. Abra o arquivo para instalar

## Alternativa: Usar Expo Go (Mais Rápido)

Se quiser testar rapidamente sem compilar:

1. Baixe o app "Expo Go" na Play Store
2. Abra o app
3. Escaneie o QR code do projeto
4. O app abrirá no Expo Go (sem precisar compilar)

## Troubleshooting

### "Build Failed"
- Verifique se o projeto está correto
- Tente novamente em alguns minutos
- Contate o suporte Expo

### "APK não instala"
- Ative "Instalar apps desconhecidos" nas configurações
- Verifique se tem espaço no celular
- Tente desinstalar versões antigas

### "Erro de permissão"
- Faça login novamente em https://expo.dev
- Verifique se tem acesso ao projeto

## Depois de Instalar

1. Abra o app Double
2. Ele carregará o site automaticamente
3. Se precisar instalar outro APK, o diálogo de permissão aparecerá
4. Clique em "Abrir Configurações" para permitir fontes desconhecidas
5. Volte ao app e tente novamente
